# SampleSize

**Valid for**: CANoe DE • CANoe4SW DE

## Property

SampleSize

## Description

Specifies the size of each sample, in bytes, in a media type.

## Data Type

dword

## Remarks

This property is valid only if the [FixedSizeSamples](CAPLfunctionFixedSizeSamples.md) property is **TRUE**.
