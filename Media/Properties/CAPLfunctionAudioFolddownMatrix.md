# AudioFolddownMatrix

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE • CANoe4SW DE

## Property

AudioFolddownMatrix

## Description

Specifies how an audio decoder should transform multichannel audio to stereo output. This process is also called fold-down.
