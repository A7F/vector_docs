# StopMacroFile

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
void StopMacroFile(dword handle);
```

## Description

Stops the macro from playing with the `handle` handle.

## Parameters

- **handle**  
  Handle of the started macro. The handle is returned from the [StartMacroFile](CAPLfunctionStartMacroFile.md) function when starting the macro.
