# GetMeasurementIndex

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```
long GetMeasurementIndex()
```

## Description

Returns the current measurement index. The measurement index can be set in the global options and is incremented on each measurement start.
