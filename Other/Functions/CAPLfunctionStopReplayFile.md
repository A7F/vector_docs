# StopReplayFile

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
void StopReplayFile(dword handle);
```

## Description

Stops the replay file from playing with the `handle` handle.

## Parameters

- **handle**: Handle of the started replay file. The handle is returned from the [StartReplayFile](CAPLfunctionStartReplayFile.md) or [StartDOReplayFile](CAPLfunctionStartDOReplayFile.md) function when starting the replay file.
