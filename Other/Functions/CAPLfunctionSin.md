# sin

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
double sin(double x);
```

## Description

Calculates sine of x.

## Parameters

Value in radians whose sine is to be calculated.

## Return Values

Sine of x.

## Example

```plaintext
double x;
x = sin(PI); // Result 0
```
