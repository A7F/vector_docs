# abs

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
long abs(long x);
double abs(double x);
```

## Description

Returns the absolute value.

## Parameters

Value whose absolute value is to be returned.

## Return Values

Absolute value of x.

## Example

```plaintext
long x;
x = abs(15); // Result: 15
```
