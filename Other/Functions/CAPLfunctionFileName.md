# fileName

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
void fileName();
```

## Description

Output of the CAPL program name in the Write Window. Helpful for debugging purposes.

—

## Return Values
