# ARILControlStart

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe4SW DE

**Note**  
This function should not be used in the **on PreStart** event procedure.

## Function Syntax

`long ARILControlStart ()`

## Description

When the IL is started, then it is fully operating and sending.
