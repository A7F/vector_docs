# ARILControlStop

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe4SW DE

**Note**  
This function should not be used in the **on PreStart** event procedure.

## Function Syntax

```plaintext
long ARILControlStop()
```

## Description

Stops sending of PDUs.
