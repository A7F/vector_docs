# CANstressStopTrigger

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE

## Function Syntax

```
long CANstressStopTrigger();
```

## Description

Deactivates the CANstress software triggers if this is still active.

## Parameters

—

## Return Values

- **0**: On successful call.
