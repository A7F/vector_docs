# canGetDataLength

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
long canGetDataLength(message msg);
```

## Description

Returns the valid payload length of the can message.

## Parameters

- **msg**: CAN message

## Return Values

0 — 64
