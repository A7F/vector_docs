[Valid for](../../../../Shared/FeatureAvailability.md): CANoe DE • CANoe4SW DE

## Function Syntax

```
long SomeIpTCPListen(dword aepHandle)
```

## Description

## Parameters

- **aepHandle**: an application endpoint

## Return Values
