# SetMapObjectVisible

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE • CANoe:lite DE • CANoe4SW DE • CANoe4SW:lite DE

## Function Syntax

```plaintext
long SetMapObjectVisible ( long handle, int visible );
```

## Description

Sets the visibility of a map object.

## Parameters

- **handle**: Reference of the map object.
- **visible**:
  - 0 = not visible
  - 1 = visible

## Return Values

- **0**: Success
- **≠0**: The set went wrong.
