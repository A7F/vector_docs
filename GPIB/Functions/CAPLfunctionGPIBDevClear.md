# GPIBDevClear

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE

## Function Syntax

```plaintext
long GPIBDevClear (long deviceDescriptor);
```

## Description

Resets a device.

## Parameters

- **deviceDescriptor**: Device ID

## Return Values

- [Status](../CAPLfunctionsGPIBStatus.md)
- **-1**: On error, if no GPIB driver is available
