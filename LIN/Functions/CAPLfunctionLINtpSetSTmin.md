# LINtp_SetSTmin

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE • CANoe4SW DE

## Function Syntax

```plaintext
void LINtp_SetSTmin(dword STmin_ms);
```

## Description

Sets minimum separation time to use.

## Parameters

- **STmin_ms**: 
  - `[0, 127]`: Separation time in milliseconds
  - `other`: 127 ms will be used
