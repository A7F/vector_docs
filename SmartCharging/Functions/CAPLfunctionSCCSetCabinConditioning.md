# SCC_SetCabinConditioning

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE  •  CANoe4SW DE

## Function Syntax

`long SCC_SetCabinConditioning (long CabinConditioning)`

## Description

The function sets the cabin conditioning flag.

## Parameters

- **CabinConditioning**
  - 1 = conditioning on
  - 0 = conditioning off

## Return Values

- **0**
  - Not successful
- **1**
  - Successful
