## Function Syntax

```plaintext
long SCC_SetMeterReading( float CurrentMeter )
```

## Description

Sets the current consumption in the charge point.

## Parameters

- **CurrentMeter**: Meter reading to be set in Wh.

## Return Values

- **0**: Not successful
- **1**: Successful
