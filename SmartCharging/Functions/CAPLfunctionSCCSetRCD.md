### Function Syntax

```plaintext
long SCC_SetRCD ( long RCD )
```

### Description

The function sets the residual current device.

### Parameters

- **RCD**
  - Open, error = 1
  - Closed, no error = 0

### Return Values

- **0**: Not successful
- **1**: Successful
