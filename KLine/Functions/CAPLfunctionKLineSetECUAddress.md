# KLine_SetECUAddress

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE

## Function Syntax

```
long KLine_SetECUAddress(Byte ecuAddress)
```

## Description

Sets the address of the ECU.

## Parameters

- **ecuAddress**: Address of the ECU.

## Return Values

On success 0, otherwise a value less than 0.
