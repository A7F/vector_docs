# KLine_SetP4Tester

[Valid for](../../../Shared/FeatureAvailability.md):  CANoe DE

## Function Syntax

```
long KLine_SetP4Tester(dword P4_us)
```

## Description

Sets the interbyte time of a request.

## Parameters

- **P4_us**: P4 in us  
  Value range: 0 – 650000.

## Return Values

On success 0, otherwise a value less than 0.
