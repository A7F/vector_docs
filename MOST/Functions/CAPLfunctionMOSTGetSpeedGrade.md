# mostGetSpeedGrade

[Valid for](../../../Shared/FeatureAvailability.md): CANoe DE

## Function Syntax

```
long mostGetSpeedGrade(long channel);
```

## Description

Returns the configured speed grade of a MOST channel.

## Parameters

- **channel**: Application channel number.

## Return Values

- **Speed grade**:
  - 1: MOST25
  - 2: MOST50
  - 3: MOST150
